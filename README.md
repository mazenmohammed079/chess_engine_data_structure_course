# ♟️ Chess Engine GUI

A beautiful web-based chess GUI powered by a custom C++ engine and Streamlit.

## Features
- Play chess with a C++ engine
- Move history tracking
- Undo/Redo functionality
- Beautiful UI

## Play Online
[Live Demo](https://your-app-name.streamlit.app)

## Local Setup
```bash
pip install -r requirements.txt
streamlit run app.py
```
